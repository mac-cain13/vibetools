# Use PowerShell when needed

You're working on Windows apps/projects from inside WSL, so you might need to use powershell.exe to run tools like MSBuild.exe and verify things build or launch apps etc.

# Available tools

These tools and their dependencies are installed and available to use if you might need them: ripgrep jq fd fzf bat tree yq htmlq gh git-delta hyperfine watch tldr pandoc xcbeautify imagemagick ffmpeg
